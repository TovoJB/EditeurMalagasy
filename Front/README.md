# 🎨 Malagasy Morpheme — Frontend

Ce dossier est l'interface utilisateur (UI) permettant de tester et visualiser la segmentation morphologique de la langue malgache.

---

## 🛠️ Rôle et Technologies

Le frontend communique avec l'API Backend pour offrir une expérience interactive et intuitive de la segmentation.

### Fonctionnalités prévues :
1.  **Zone de test Interactive** : Entrée d'un mot et affichage instantané de sa segmentation.
2.  **Visualisation de la Racine** : Mise en évidence visuelle des morphèmes de racine (en utilisant les crochets du modèle).
3.  **Visualisation des Statistiques** : Intégration des dashboards (`training_curves.png`, `vocabular.png`) pour les administrateurs.

### Stack Technologique suggérée :
- **React / Vue.js** : Pour la réactivité des composants.
- **Tailwind CSS / Lucide Icons** : Pour une apparence moderne, sombre et premium.
- **Axios / React Query** : Pour la communication asynchrone avec le backend.

---

## 🎨 Design System

L'interface doit inspirer confiance par un design **Premium et Épuré** :
- **Mode Sombre** par défaut (Palette de couleurs Deep Navy & Slate).
- **Typographie moderne** (ex: Outfit ou Inter) pour une lisibilité maximale.
- **Micro-animations** lors de la segmentation d'un mot.
